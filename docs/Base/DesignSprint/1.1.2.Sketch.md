# 1.1.2 Sketch

## Introdução

Tal como preconizado pela Google Ventures, a fase de Sketch vem logo após o entendimento do problema e a escolha de um alvo para se trabalhar na sprint. Durante esta fase, o objetivo é focar em soluções para o problema e, como o nome sugere, cada pessoa irá rascunhar priorizando o pensamento crítico sobre deixar artístico. Conforme sugerido por Milene, haverá uma adaptação do Sketch para que o artefato a ser entregue por todos os membros do grupo durante essa parte seja os Rich Pictures individuais, com cada um propondo um modelo diferente.
