Tal como preconizado pela Google Ventures (2019), a fase de Sketch vem logo após o entendimento do problema e a escolha de um alvo para se trabalhar na sprint. Durante esta fase, o objetivo é focar em soluções para o problema e, como o nome sugere, cada pessoa irá rascunhar priorizando o pensamento crítico sobre deixar artístico.
